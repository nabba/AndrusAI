# Advanced Ecological Data Cleaning

## Techniques
1. **Missing Data Imputation**
   - Time-series interpolation
   - Species-specific median imputation
   - Environmental variable regression imputation

2. **Outlier Detection**
   - Modified Z-score for skewed distributions
   - Seasonal decomposition for time-series
   - Mahalanobis distance for multivariate data

3. **Data Normalization**
   - Log transformation for abundance data
   - Min-max scaling for environmental variables
   - Box-Cox transformation for non-normal distributions
