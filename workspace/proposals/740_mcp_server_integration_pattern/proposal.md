# Proposal #740: mcp_server_integration_pattern

**Type:** skill  
**Created:** 2026-04-21T13:59:12.328761+00:00  

## Why this is useful

PROBLEM: The team has MCP (Model Context Protocol) infrastructure but no MCP servers are currently connected. Agents lack guidance on discovering and adding external capabilities. Many tasks that could leverage external services (databases, communication platforms, calendars) fail or require manual workarounds. SOLUTION: Create a skill documenting the MCP integration workflow: 1) How to search for relevant MCP servers, 2) How to evaluate server suitability, 3) Environment variable requirements, 4) Connection testing procedures, 5) Common server categories and use cases. This enables any crew to extend capabilities on-demand rather than requiring central tool additions.
