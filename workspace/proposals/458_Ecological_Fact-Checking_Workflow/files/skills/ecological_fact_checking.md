# Ecological Fact-Checking Workflow

1. **Claim Identification**: Extract factual claims from content
2. **Source Verification**: Cross-reference with:
   - Peer-reviewed journals (via DOI lookup)
   - Government environmental reports
   - Reputable scientific databases
3. **Confidence Scoring**: Rate evidence quality (1-5 scale)
4. **Discrepancy Flagging**: Highlight unsupported claims

Tools needed: DOI resolver, academic database API connectors
