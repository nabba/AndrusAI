# Database Design and SQL Mastery

## Overview
This skill covers the fundamentals and advanced concepts of database design and SQL. It enables agents to create efficient, scalable, and secure data storage solutions.

## Relational Database Fundamentals
- Tables, rows, columns
- Primary keys, foreign keys
- Data types
- Normalization (1NF, 2NF, 3NF, BCNF)
- ACID properties

## SQL Queries
- SELECT, INSERT, UPDATE, DELETE
- JOINs (INNER, LEFT, RIGHT, FULL)
- Subqueries
- Common Table Expressions (CTEs)
- Window functions
- Aggregate functions
- GROUP BY, HAVING

## Performance Tuning
- Indexing strategies (B-tree, hash, GiST, etc.)
- Query optimization (EXPLAIN ANALYZE)
- Avoiding N+1 queries
- Partitioning
- Caching strategies

## Transactions and Concurrency
- Transaction isolation levels
- Deadlocks
- Optimistic vs pessimistic locking

## Database Administration
- User management and privileges
- Backups and restores
- Replication
- Connection pooling (e.g., pgBouncer)
- Monitoring (pg_stat_statements)

## NoSQL Databases
- When to use NoSQL vs SQL
- Document stores (MongoDB)
- Key-value stores (Redis)
- Column-family (Cassandra)
- Graph (Neo4j)

## ORM Best Practices
- Avoiding lazy loading pitfalls
- Using select_related/prefetch_related in Django
- Managing migrations
- Handling database schema changes

## Security
- SQL injection prevention (parameterized queries)
- Encryption at rest and in transit
- Row-level security

## Tools
- psql, sqlite3, mysql CLI
- GUI tools (pgAdmin, DBeaver)
- Migration tools (Alembic, Flyway)

- Books: 'SQL Antipatterns', 'Database Design for Mere Mortals'
