# Proposal #615: Task Decomposition and Multi-Step Planning Skill

**Type:** skill
**Created:** 2026-04-17T18:25:53.855074+00:00

## Description

Problem: The team's existing skills are heavily concentrated in the ecological domain (40+ ecological skills) with almost no skills for general-purpose task planning, decomposition, or orchestration. When faced with complex multi-step tasks outside ecology, the team lacks structured approaches for breaking problems down, estimating complexity, choosing the right crew, or recovering from failures. The team memory shows research tasks varying wildly in completion time (7s to 165s) with no apparent optimization strategy. Solution: Add a comprehensive task decomposition and planning skill that provides frameworks for analyzing incoming tasks, breaking them into sub-tasks, selecting appropriate crews, and handling failures gracefully.
