# Advanced Search Filtering

## Techniques
1. **Domain Filtering**: Use `site:` operator to limit results to specific domains.
2. **Date Filtering**: Use `before:` and `after:` to filter based on publication date.
3. **Content Type**: Use `filetype:` to filter results by content type (e.g., PDF, DOC).

## Examples
- `site:example.com` - Only results from example.com
- `after:2023-01-01` - Results published after January 1, 2023
- `filetype:pdf` - Only PDF files
