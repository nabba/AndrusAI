## Automated Fact-Checking Pipeline

System for verifying ecological claims through multi-source validation.

### Workflow:
1. Research crew identifies claim needing verification
2. Coding crew develops source triangulation algorithm
3. Writing crew formats verified information

### Techniques:
- Claim decomposition into verifiable components
- Source credibility scoring
- Discrepancy resolution protocols

### Implementation:
- Requires modification to research crew's search synthesis
- New team memory fields for verification metadata
