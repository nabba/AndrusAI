# Geospatial Analysis for Ecological Data

## Core Capabilities
- Process GeoJSON, Shapefile, and raster data
- Perform spatial joins and overlays
- Create heatmaps and choropleth visualizations
- Calculate spatial statistics

## Required Libraries
- Geopandas
- Rasterio
- Folium/Leaflet

## Common Workflows
1. Import geospatial data
2. Clean and project data
3. Perform spatial analysis
4. Visualize results
5. Export for reporting
