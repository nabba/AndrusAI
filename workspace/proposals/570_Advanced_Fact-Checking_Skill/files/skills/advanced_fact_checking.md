# Advanced Fact-Checking for Ecological Content

## Problem Statement
Ecological research and reporting often rely on web-sourced data, which can be inaccurate or outdated. Without a systematic fact-checking process, the team risks propagating misinformation.

## Solution
1. **Source Evaluation**: Use tools like Media Bias/Fact Check to assess source credibility.
2. **Cross-Referencing**: Verify facts against at least three independent sources.
3. **Temporal Analysis**: Check the publication date to ensure data relevance.
4. **Expert Consultation**: Flag uncertain data for human review.

## Workflow
1. Perform initial web search.
2. Apply source evaluation criteria.
3. Cross-reference facts.
4. Document verification steps.
5. Escalate unresolved discrepancies.
