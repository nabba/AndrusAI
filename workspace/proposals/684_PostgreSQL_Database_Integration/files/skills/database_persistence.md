# Database Persistence Skill

## Why This Matters
- Enables storing research findings, user data, and long-term memory
- Allows SQL queries for complex data analysis
- Supports multi-session continuity

## Available Tools
- Supabase MCP server: remote PostgreSQL with admin tools
- Neon MCP: managed PostgreSQL with branching

## Usage Pattern
1. Store research results with structured schemas
2. Query across large datasets
3. Track experiment history and outcomes
4. Build knowledge graphs and relationship data

## Example Tasks This Unlocks
- Long-term user preference memory
- Historical trend analysis
- Structured data validation and querying
- Automated report generation from stored data
