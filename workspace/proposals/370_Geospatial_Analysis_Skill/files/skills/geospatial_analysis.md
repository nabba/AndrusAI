# Geospatial Analysis for Ecological Data

## Core Competencies:
1. Coordinate system transformations (WGS84 to UTM)
2. Shapefile and GeoJSON processing
3. Raster data analysis (NDVI, elevation models)
4. Spatial joins and overlays
5. Distance and buffer calculations

## Recommended Tools:
- Geopandas for vector data
- Rasterio for raster data
- Folium for interactive maps

## Common Ecological Applications:
- Habitat range mapping
- Deforestation tracking
- Watershed analysis
- Species distribution modeling
