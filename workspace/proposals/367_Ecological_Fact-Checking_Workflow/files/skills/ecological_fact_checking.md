# Ecological Fact-Checking Protocol

1. **Source Verification**: Check author credentials and publication credibility
2. **Data Cross-Referencing**: Compare with peer-reviewed studies
3. **Temporal Analysis**: Verify if information is current
4. **Expert Consultation**: Flag for human review when uncertainty >20%
5. **Bias Detection**: Analyze language for sensationalism

Tools: web_search, web_fetch, memory_retrieve
