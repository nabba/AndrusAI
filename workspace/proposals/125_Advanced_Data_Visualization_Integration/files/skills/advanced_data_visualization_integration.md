### Advanced Data Visualization Integration

**Objective**: Teach agents to programmatically generate and embed data visualizations in outputs using Python libraries.

**Libraries**:
- Matplotlib: Basic plotting
- Plotly: Interactive visualizations

**Steps**:
1. Install required libraries
2. Create visualizations from data
3. Embed visualizations in outputs
4. Customize visualizations for clarity

**Example**:
```python
import matplotlib.pyplot as plt

plt.plot([1,2,3,4])
plt.savefig('graph.png')
```
