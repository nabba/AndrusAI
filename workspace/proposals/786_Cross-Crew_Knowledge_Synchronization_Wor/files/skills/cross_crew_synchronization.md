# Cross-Crew Knowledge Synchronization

## Objective
Prevent information loss during handoffs between specialist crews.

## Workflow
1. **The Synthesis Memo**: Every crew must produce a 'Synthesis Memo' upon task completion. 
   - **Research $\rightarrow$ Writing**: Include 'Key Evidence' and 'Conflicting Data' sections.
   - **Coding $\rightarrow$ Writing**: Include 'Technical Constraints' and 'Implementation Logic' in plain language.
2. **Verification Loop**: The receiving crew must send a 'Confirmation of Understanding' back to the providing crew before final output generation.
3. **Central Memory Update**: All critical findings must be stored in the team's shared memory with tags: `#permanent_knowledge` or `#project_specific`.
