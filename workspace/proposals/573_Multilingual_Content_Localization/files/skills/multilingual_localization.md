# Multilingual Content Localization

## Key Techniques
1. **Transcreation**: Adapt messaging while preserving intent (e.g., idioms for regional relevance).
2. **Cultural Context Mapping**: Align examples with local ecosystems (e.g., mangroves in Southeast Asia).

## Tools
- LibreTranslate API wrapper for open-source translation.
- Glossary of standardized ecological terms per language.
