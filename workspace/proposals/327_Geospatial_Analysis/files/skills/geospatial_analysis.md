# Geospatial Analysis for Ecological Teams

## Core Competencies
- Coordinate system interpretation (WGS84, UTM)
- Raster vs vector data handling
- Basic operations: buffers, intersects, overlays
- Environmental indices calculation (NDVI, LST)

## Recommended Tools
- GeoPandas for vector analysis
- Rasterio for satellite imagery
- Earth Engine API for large datasets

## Workflow Integration
1. Ingest geospatial data from APIs
2. Process using coordinate-aware methods
3. Visualize with contextually appropriate projections
