# Proposal #628: Cross-Crew Validation Workflow

**Type:** skill
**Created:** 2026-04-18T16:05:11.178821+00:00

## Description

There is a potential gap between the 'coding' crew (producing data/models) and the 'writing' crew (interpreting them). The team lacks a formal 'Review & Validate' loop where the writing crew can challenge the data interpretation before the final report is generated. This skill introduces a systematic peer-review protocol.
