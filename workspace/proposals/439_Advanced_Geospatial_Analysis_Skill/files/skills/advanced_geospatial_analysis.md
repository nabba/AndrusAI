# Advanced Geospatial Analysis

## Introduction
Handling geospatial data is essential for ecological impact assessments. This module covers the use of Python libraries like GeoPandas and Shapely for effective geospatial data analysis.

## Topics Covered
- Introduction to GeoPandas
- Geospatial data manipulation
- Spatial joins and overlays
- Visualization of geospatial data

## Practical Exercises
1. Load a shapefile and plot it using GeoPandas.
2. Perform a spatial join between two datasets.
3. Create a choropleth map for ecological data.
