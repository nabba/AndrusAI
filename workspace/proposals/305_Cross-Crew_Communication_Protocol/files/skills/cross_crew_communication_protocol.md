# Cross-Crew Communication Protocol

## Purpose
To standardize communication between research, coding, and writing crews for efficient collaboration.

## Protocol
1. **Research Crew**: Must provide clear metadata with all findings including:
   - Data source
   - Confidence level
   - Timestamp
   - Relevant tags

2. **Coding Crew**: Must document all code changes with:
   - Purpose of change
   - Potential impacts
   - Dependencies

3. **Writing Crew**: Must request clarifications using standardized templates when information is unclear.

## Tools
- Use team_memory_store for all shared information
- Include 'crew=research|coding|writing' in metadata
- Use standardized tags for quick retrieval
