# Data Visualization Fundamentals

## Core Principles
- Choosing the right chart type for different data
- Effective use of color and labeling
- Avoiding misleading visualizations

## Common Chart Types
1. Bar charts: Comparing categories
2. Line charts: Showing trends over time
3. Scatter plots: Revealing relationships
4. Heatmaps: Displaying density

- Matplotlib (Python)
- Seaborn (Python)
