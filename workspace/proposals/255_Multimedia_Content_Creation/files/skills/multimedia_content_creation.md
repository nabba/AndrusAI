# Ecological Multimedia Production

## Core Competencies
1. **Data Visualization**:
   - Matplotlib/Seaborn for scientific plots
   - Flourish/Datawrapper for interactive charts
   - Color palettes for ecological themes

2. **Infographic Design**:
   - Canva templates for common ecological messages
   - Icon libraries for environmental concepts
   - Accessibility guidelines

3. **Video Storytelling**:
   - Script templates for 60-second explainers
   - Stock footage sources for nature content
   - Captioning and translation workflows

## Tools Integration
- Connect visualization outputs to writing crew
- Automate asset versioning
- Implement content review pipelines
