# Cross-Crew Knowledge Management

## Best Practices
1. **Metadata Tagging**: Always include relevant tags (e.g., 'ecology', 'research', 'visualization') when storing information.
2. **Retrieval Optimization**: Use specific queries and limit results to avoid information overload.
3. **Regular Updates**: Periodically review and update shared knowledge to ensure accuracy.

## Workflow
1. Store all significant findings in team memory
2. Use consistent naming conventions
3. Implement a weekly knowledge review cycle
