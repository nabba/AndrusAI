# Proposal #710: PDF Document Processing

**Type:** skill

## Why this is useful

Problem: The team cannot extract content from PDF files, which are ubiquitous in research tasks (academic papers, reports, contracts, invoices). Current tools only handle web pages and plain text. Solution: A skill that teaches agents how to extract text, tables, and metadata from PDFs using Python libraries like PyMuPDF, pdfplumber, and pypdf. Includes patterns for OCR when needed and structured data extraction from forms.
