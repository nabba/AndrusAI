# Geospatial Analysis for Ecological Data

## Core Competencies
- Processing GeoJSON and shapefile data
- Spatial joins and overlays
- Heatmap generation
- Watershed analysis
- Habitat fragmentation metrics

## Recommended Tools
- GeoPandas for vector data
- Rasterio for satellite imagery
- Folium/Leaflet for interactive maps
- PostGIS for complex queries

## Workflow Integration
1. Standardize coordinate reference systems
2. Automate spatial joins with ecological datasets
3. Generate standardized map outputs
4. Integrate with existing visualization pipelines
