# Rapid Fact-Checking Techniques

## Overview
This document outlines methods for quickly verifying ecological data and claims using web search and data validation.

## Steps
1. **Keyword Identification**: Extract key terms from the claim.
2. **Web Search**: Use advanced web search techniques to find verifying sources.
3. **Data Validation**: Check the consistency and credibility of the information found.
4. **Report Findings**: Summarize the verification results for team use.
