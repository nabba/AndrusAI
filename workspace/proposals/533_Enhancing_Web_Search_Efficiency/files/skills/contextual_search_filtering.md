## Contextual Search Filtering

**Objective:** To refine web search queries based on context and previous findings.

### Steps:
1. **Analyze Previous Results:** Before initiating a new search, analyze results from previous searches stored in team_memory.
2. **Contextual Keywords:** Extract keywords and context from these results to refine new search queries.
3. **Advanced Filters:** Use advanced search operators and filters to narrow down results based on contextual keywords.
4. **Continuous Learning:** Update search strategies based on the relevance of returned results to continuously improve accuracy.
