# Geospatial Analysis for Ecological Teams

## Core Competencies
1. GIS Fundamentals
   - Coordinate systems
   - Spatial data formats (GeoJSON, Shapefiles)
   - Basic QGIS operations

2. Spatial Analysis Techniques
   - Hotspot analysis
   - Spatial autocorrelation
   - Buffer zones and proximity analysis

3. Map Visualization
   - Choropleth mapping
   - Heatmaps
   - Time-series spatial animations

## Recommended Tools
- GeoPandas (Python library)
- Leaflet.js (for web maps)
- Google Earth Engine (for satellite data)
