# Data Visualization Skill

## Problem
The team has specialized data visualization skills but lacks general capabilities.

## Solution
This skill provides general methods for visualizing data using libraries like Matplotlib, Seaborn, and Plotly.

### Tools Required
1. Matplotlib
2. Seaborn
3. Plotly

## Workflow
1. Load data
2. Choose visualization type
3. Create plot
4. Customize and export
