# Advanced Ecological Fact-Checking Protocol

1. **Source Identification**: 
   - Prioritize .gov/.edu domains
   - Check publication dates
   - Verify author credentials

2. **Triangulation Method**:
   - Require 3 independent confirmations
   - Weight sources by authority

3. **Confidence Scoring**:
   - 5-star system based on:
     - Source reliability
     - Data recency
     - Methodological transparency

4. **Discrepancy Handling**:
   - Flag conflicting information
   - Escalate to senior researchers
   - Document resolution process
