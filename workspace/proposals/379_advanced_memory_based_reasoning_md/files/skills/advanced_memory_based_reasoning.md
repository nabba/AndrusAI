## Advanced Memory-Based Reasoning

This skill focuses on optimizing the use of memory storage and retrieval tools to enhance team decision-making and knowledge synthesis.
### Key Techniques:
1. **Contextual Storage:** Store information with relevant metadata for easier retrieval.
2. **Hierarchical Retrieval:** Use structured queries to retrieve relevant information efficiently.
3. **Synthesis:** Combine retrieved information from different sources for comprehensive insights.
### Example:
```json
{
  "text": "Ecological data analysis shows increasing deforestation rates...",
  "metadata": "source=web_search, query=deforestation, date=2023-10-01"
}
```
