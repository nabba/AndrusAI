# Collaborative Document Editing Workflow

## Overview
This skill streamlines cross-crew collaboration on documents (e.g., reports, policy briefs) using version control and real-time editing tools.

## Key Techniques
1. **Git Basics**: Learn to create branches, commit changes, and resolve merge conflicts.
2. **VS Code Live Share**: Use for real-time collaborative editing with permissions.
3. **Markdown Templates**: Standardize document structures for consistency.

## Example Workflow
1. Research crew drafts findings in a Markdown file.
2. Writing crew reviews and edits via a Git branch.
3. Final version is merged and published as a PDF.
