# Ecological Data Cleaning and Preprocessing

## Handling Missing Data
- **Techniques**: Imputation (mean, median), deletion, or prediction models.
- **Tools**: Pandas for simple imputation, Scikit-learn for advanced methods.

## Outlier Detection
- **Techniques**: Z-score, IQR, visual methods (box plots).
- **Tools**: Scipy for statistical methods, Matplotlib for visualization.

## Normalization
- **Techniques**: Min-max scaling, standardization.
- **Tools**: Scikit-learn's preprocessing module.
