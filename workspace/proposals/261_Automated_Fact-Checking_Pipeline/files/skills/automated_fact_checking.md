# Automated Fact-Checking Workflow

## Process:
1. Extract claims from content
2. Search trusted ecological databases (NCBI, IUCN, etc.)
3. Compare claims against scientific consensus
4. Generate verification report

## Implementation:
- Use named entity recognition for ecological terms
- Maintain whitelist of trusted sources
- Implement confidence scoring system

## Output:
- Verified claims with citations
- Flagged questionable claims
- Suggested corrections
